## 相似工作1: HeR-DRL (2024年3月) ⭐⭐⭐⭐⭐ 最相似
**论文**: "Heterogeneous Relational Deep Reinforcement Learning for Decentralized Multi-Robot Crowd Navigation"
**代码**: https://github.com/Zhouxy-Debugging-Den/HeR-DRL

### 相似度分析
| 维度 | 当前项目 | HeR-DRL | 相似度 |
|------|---------|---------|--------|
| 算法 | MADDPG | DRL + 异构GNN | ⭐⭐⭐ |
| 观测 | 激光雷达(38维) | 激光雷达 + 位置 | ⭐⭐⭐⭐⭐ |
| 动作 | 连续[v, ω] | 连续[v, ω] | ⭐⭐⭐⭐⭐ |
| 场景 | 室内多机器人 | 多机器人人群导航 | ⭐⭐⭐⭐ |

### 核心技术
1. **异构图神经网络（Heterogeneous GNN）**
   - 区分机器人-机器人、机器人-障碍物的不同交互
   - 使用不同的权重聚合不同类型实体的信息

2. **奖励设计**
   ```python
   # 参考HeR-DRL的奖励函数
   reward = reward_goal + reward_collision + reward_progress + reward_safety
   
   # 进步奖励（与你相同）
   reward_progress = (prev_dist - curr_dist) * scale_factor
   
   # 安全奖励（避免危险区域）
   d_min = min_distance_to_obstacles
   if d_min < 0.2:
       reward_safety = -0.15
   elif d_min < 0.4:
       reward_safety = (d_min - 0.2) * 0.25 - 0.15  # 线性递增
   else:
       reward_safety = 0.0
   ```

3. **解决打转问题的方法**
   - **动作平滑惩罚**: 相邻时刻动作变化太大会被惩罚
   ```python
   # 动作变化惩罚
   action_change = abs(action[t] - action[t-1])
   reward_smoothness = -0.01 * action_change.sum()
   ```
   - **无效旋转检测**: 持续高角速度 + 低线速度
   ```python
   if abs(angular_vel) > 0.5 and abs(linear_vel) < 0.1:
       reward += -0.5  # 打转惩罚
   ```

## 相似工作3: Loc-FACMAC (2025年3月) ⭐⭐⭐⭐
**论文**: "Learning Multi-Robot Coordination through Locality-Based Factorized Multi-Agent Actor-Critic"

### 相似度分析
| 维度 | 当前项目 | Loc-FACMAC | 相似度 |
|------|---------|------------|--------|
| 算法 | MADDPG | FACMAC | ⭐⭐⭐⭐ |
| 观测 | 激光雷达 | 传感器观测 | ⭐⭐⭐⭐ |

| 场景 | 多机器人 | 多机器人协作 | ⭐⭐⭐⭐⭐ |

### 核心技术
1. **局部性Critic（Locality-based Critic）**
   - 每个机器人只关注邻近机器人的状态和动作
   - 减少维度灾难，提高训练效率

2. **依赖图构建**
   - 基于距离构建机器人依赖关系
   ```python
   # 构建局部依赖（参考Loc-FACMAC）
   def get_local_neighbors(self, robot_id, threshold=3.0):
       neighbors = []
       for j in range(self.number_of_robots):
           if j != robot_id:
               dist = math.hypot(
                   self.current_pose_x[robot_id] - self.current_pose_x[j],
                   self.current_pose_y[robot_id] - self.current_pose_y[j]
               )
               if dist < threshold:
                   neighbors.append(j)
       return neighbors
   ```

3. **解决探索问题**
   - 使用参数空间噪声（Parameter Space Noise）替代动作空间噪声
   ```python
   # 参数噪声：直接给网络权重加噪声
   for param in actor.parameters():
       param.data += torch.randn_like(param) * noise_std
   ```

### 建议应用（可选，较复杂）
当前MADDPG已经是集中训练分散执行，Loc-FACMAC的改进需要修改Critic结构，工作量大。如果训练效率成为瓶颈可以考虑。

```python
# agent.py

class Agent:
    def __init__(self, ..., initial_noise=0.3, noise_decay=0.9995, min_noise=0.05):
        # ... 现有代码 ...
        self.noise_std = initial_noise
        self.noise_decay = noise_decay
        self.min_noise = min_noise
    
    def choose_action(self, observation, explore=True):
        # ... 现有代码获取actions_np ...
        
        if explore:
            noise = np.random.normal(0, self.noise_std, size=self.n_actions)
            action = np.clip(actions_np + noise, -1., 1.)
        else:
            action = actions_np
        
        return action
    
    def decay_noise(self):
        """每个episode结束后调用"""
        self.noise_std = max(self.min_noise, self.noise_std * self.noise_decay)

# maddpg_main.py 中，每个episode结束后：
for agent in maddpg_agents.agents:
    agent.decay_noise()
```
